# Frontend

```
-- Login / Register
                    -- Dashboard -
                                 - Leer QR
                                 - Depositar    ->  Generar QR
                                                ->  Generar enlace
                                                ->  Datos de Cuenta
                                                ->  Deposito con Mercado pago - Pronto
                                 - Transferencia
                                                -> Solo ingresamos dos elementos
                                                    -> Monto
                                                    -> Wallet o Alias
```
